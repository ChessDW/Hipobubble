# 🫧 Hipobubble Web Application

**Hipobubble** es una plataforma web orientada al sector de cuidado personal y productos naturales, desarrollada bajo un enfoque de **arquitectura de frontend escalable**, priorizando la experiencia de usuario (UX), la accesibilidad y la internacionalización.

## 🚀 Technical Highlights
* **Internationalization (i18n):** Sistema de traducción dinámica integrado para soporte en español, inglés y francés.
* **Theming Engine:** Implementación de modo oscuro/claro con persistencia de estado para el usuario.
* **Performance Focused:** Diseño optimizado con *lazy loading* de assets y animaciones fluidas (Scroll Progress Bar).
* **Responsive Architecture:** Diseño *mobile-first* con navegación adaptativa (menú hamburguesa).
* **SEO Optimized:** Etiquetas meta estratégicas para posicionamiento local en Costa Rica.

## 🛠️ Stack & Technologies
* **Core:** HTML5, CSS3, Vanilla JavaScript.
* **Iconography:** FontAwesome 6.0.
* **Typography:** Google Fonts (Playfair Display & Open Sans).
* **Deployment:** Netlify.

## 🧠 Project Architecture
El proyecto está estructurado para facilitar la escalabilidad:
- `/css`: Hojas de estilo modulares.
- `/js`: Lógica de negocio, gestión del modal de configuración y manejo de estados.
- `/media`: Gestión de assets optimizados (formatos .webp y .ico).
